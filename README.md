# dicebear_avatars.py
Web-API for [avatars.dicebear.com](https://avatars.dicebear.com) an avatar library for designers and developers

## Example
```python
from dicebear_avatars import DiceBearAvatars

db_avatars = DiceBearAvatars()
db_avatars.get_avatar(
  sprite="", seed="",  background="", mood="", format="")
```
